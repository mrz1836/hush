```bash

magex format:fix && magex lint && magex test:race

# Coverage on the new package, scoped to the state file.

# Direct race-detector run on the concurrent test.

# Confirm zero test-time goroutine leaks.
go test -run TestStore_NoSideEffects ./internal/supervise/
```

Expected:
- `magex` gates green.
- `go test -cover` reports ≥ 95% on `internal/supervise/state.go`.
- Race detector clean.
- `TestStore_NoSideEffects` passes (zero goroutine delta).
